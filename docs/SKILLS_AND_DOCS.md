# 云文档内化为技能（规划）

后续你会在 Lark 里发一些云文档。目标能力：

1. **阅读并总结**：能阅读这些文档并按要求总结。
2. **内化为技能**：把文档内容当作知识库，别人问相关问题时能从文档里提取信息回答。
3. **按模板生成新文档**：文档里有模板时，可以根据用户给的新信息，按模板格式生成新文档。

## 在 Cursor 里（当前对话）

- 已在 `.cursor/rules/lark-docs-as-skills.mdc` 里加了一条规则。
- 你把云文档内容粘贴到对话里（或说明链接/导出内容）后，我会：
  - 总结文档；
  - 把文档当知识用，回答相关问题；
  - 若有模板，根据你给的新信息按模板生成新文档。

## 在 Lark 机器人里（后续扩展）

计划在 `app.py` 里增加：

- **知识来源**：把文档要点/FAQ 存到 `docs_knowledge/`（或配置文件），供机器人回复时查询。
- **意图**：例如用户问「XX 流程是什么」「根据 XX 模板帮我写一份 YY」时，匹配到「文档问答」或「按模板生成」意图，从知识库取内容或填模板后回复（或回复「已生成，请查收」+ 附件/链接逻辑，视飞书 API 能力再定）。

你发文档过来后，可以同时说「总结一下」「以后别人问 XX 就从这里答」「按这个模板，帮我生成一份内容是 YY 的文档」，我会按上面规则执行；需要做到 Lark 机器人里时，再一起改 `app.py` 和知识文件。
